Cadastra disponibilidade

UMLUseCase
Untitled :: Model1 :: Cadastra disponibilidade

Description

Identificação: UC_01 Atores: Guia Função: Cadastrar a disponibilidade e detalhes do serviço do guia Prioridade: Essencial Pré-condição: Guia deve estar logado Pós-condição: Cadastro feito com sucesso

Fluxo Principal: Entrar no sistema. Acessar a aba “Disponibilidades”. Clicar no botão “Adicionar Período”. Passar as informações solicitadas (Local de serviço, data de início, data de término e valor da diária). Finalizar o registro com o botão “Confirmar”. Guia é notificado que o cadastro foi realizado com sucesso.

Fluxo de Exceção: Entrar no sistema. Acessar a aba “Disponibilidades”. Clicar no botão “Adicionar Período”. Passar as informações solicitadas (Local de serviço, data de início, data de término e valor da diária). Finalizar o registro com o botão “Confirmar”. Sistema nega a adição. Já existir uma disponibilidade para a data informada. Guia é notificado do erro. Retorna para a tela de adição
