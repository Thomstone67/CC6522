Realiza reserva

UMLUseCase
diagrama de caso de uso :: Casos de uso :: Realiza reserva

Description

Identificação: UC_06 Atores: Cliente Prioridade: Essencial Função: Realizar a reserva do serviço para o pagamento Pré-condição: Guia ter aceito a reserva Pós-condição: Reserva feita com sucesso

Fluxo Principal: Cliente receba a notificação que o guia aceitou a reserva Cliente revê as informações Cliente confirma a reserva Reserva feita com sucesso

Fluxo de Exceção: Cliente receba a notificação que o guia aceitou a reserva Cliente revê as informações Cliente cancela a reserva Reserva cancelada Retorna ao caso de uso "Escolhe o serviço"
